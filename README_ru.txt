Мод представляет собой набор настроек для командиров, регулируемых через json:
/Lang/jscommandersmodule_settings.json


Список опций:
- см. jscommandersmodule_settings: список содержит несколько опций, каждая опция имеет описание действия и совместимости с WoG-опциями
- чтобы выключить опцию, нужно поменять значение ключа "value" с true на false, и наоборот
